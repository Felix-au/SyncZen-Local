## Welcome to SyncZen Local v1.0.0! 🏨⚡

**SyncZen Local** is a secure, local-first guest check-in and hotel data management system that runs entirely within your hotel's local area network (LAN). By hosting its own Express API server and utilizing in-memory WebAssembly SQLite, SyncZen Local keeps your front desk moving instantly—even if your internet goes down.

This initial release introduces the core administrative workstation (desktop) and check-in assistant client (mobile).

---

### 📦 Release Assets & Executables

We are releasing three prebuilt binaries for this version:

1.  **Windows Desktop Setup Installer (`SyncZen-Local-Setup-1.0.0.exe`):**
    *   *Role:* Installs the workstation application, creates desktop and Start Menu shortcuts, and automatically prompts for administrator privileges to configure local Windows Firewall inbound rules.
2.  **Windows Desktop Portable EXE (`SyncZen Local 1.0.0.exe`):**
    *   *Role:* Runs the workstation directly without an installer—ideal for quick tests or running from external storage.
3.  **Android Release APK (`app-release.apk`):**
    *   *Role:* The client assistant app to be sideloaded on staff mobile devices for scanning IDs and completing guest registration workflows.

### ✨ Key Features in v1.0.0

#### 🖥️ Workstation (Desktop Admin Panel)
*   **WASM Database Engine:** Employs pure WebAssembly SQLite (`sql.js`) for lightning-fast local queries without native binary conflicts.
*   **Auto-Persistence:** Automatically exports the in-memory database and writes it to `%APPDATA%/SyncZen Local/synczen.db` on every change.
*   **LAN Host Server:** Automatically binds and serves API routes via an embedded Express server on port `8080` (or increments if occupied).
*   **Auto-Checkout System:** Scans stay entries on startup and automatically marks overdue bookings as checked out to free up rooms.
*   **Pairing Hub:** Generates a secure cryptographic coupling token and renders a connection QR code for instant mobile integration.

#### 📱 Assistant (Mobile Client)
*   **QR Code Pairing:** Instantly couples with the desktop workstation via local WiFi by scanning the Pairing QR code.
*   **Check-In Wizard:** Renders a step-by-step guest flow (Party size ➔ details entry ➔ photo capture ➔ ID proof scanning ➔ room allocation).
*   **Offline-First Sync Engine:** If the network goes down, guest check-ins and compressed base64 images are buffered inside native `AsyncStorage`.
*   **Background Sync Loop:** A 15-second background timer continuously pings the workstation, uploading and syncing queued registrations automatically when the network is restored.

### 🔧 Installation & Sideloading Quick Guide

#### Windows Desktop Workstation
*   Download and run `SyncZen-Local-Setup-1.0.0.exe`.
*   Accept the **User Account Control (UAC)** prompt on launch to allow the inbound firewall rule to be registered (allows mobile devices to connect over WiFi).

#### Android Mobile Assistant
*   Transfer `app-release.apk` to your Android device.
*   Navigate to **Settings ➔ Apps ➔ Special app access ➔ Install unknown apps** and enable permission for your browser or file manager.
*   Locate and tap the APK file to install the **SyncZen Local** assistant.
*   Ensure both your computer and mobile device are connected to the **same WiFi router/LAN**.
