## Welcome to SyncZen Local v1.0.0! 🏨✨

**SyncZen Local** is a simple, secure, and private hotel guest registration and check-in system designed to work entirely within your hotel's local WiFi network. It does not require any internet connection, cloud databases, or active monthly subscriptions. All guest records, profile photos, and ID documents remain safe and secure on your own local office computer.

This release introduces the primary **Desktop Admin Dashboard** (for the reception workstation) and the companion **Mobile Check-In Assistant** (for staff mobile devices).

---

### 📦 Downloads & Installation Files

We are releasing three files for this version:

1.  **Windows Desktop Installer (`SyncZen-Local-Setup-1.0.0.exe`):**
    *   *Best for:* Standard reception computers. Installs the application, adds desktop shortcuts, and automatically configures your computer to allow mobile devices to connect.
2.  **Windows Desktop Portable Version (`SyncZen Local 1.0.0.exe`):**
    *   *Best for:* Running the application immediately without installing it—ideal for quick trials or running from a USB drive.
3.  **Android Mobile App (`SyncZen.apk`):**
    *   *Best for:* Installing on staff mobile phones or tablets to scan guest IDs, take photos, and register check-ins on the go.

---

### 🚀 Key Features in v1.0.0

#### 🖥️ Reception Workstation (Desktop Dashboard)
*   **Simple Setup:** Gets up and running with a single click.
*   **Live Dashboard Gauges:** Displays real-time hotel stats, occupancy levels, guest counts, and room lists.
*   **Automatic Room Clean-Up:** Automatically frees up rooms and processes check-outs for bookings whose departure dates have passed.
*   **Instant Room Manager:** Manage room numbers, pricing, room status (available, occupied, maintenance), and notes.
*   **Easy Device Pairing:** Renders a secure connection QR code on-screen. Scanning it with a mobile device instantly pairs them over WiFi.
*   **Safe Data Storage:** Automatically saves all hotel data and settings to a local file on your computer.

#### 📱 Mobile Assistant (Phone/Tablet App)
*   **WiFi Pairing:** Scan the desktop QR code to connect instantly.
*   **Step-by-Step Check-In:** Guide guests through registration easily (Select party size ➔ enter guest details ➔ snap profile photos ➔ scan ID proofs ➔ assign available rooms ➔ confirm stay).
*   **Offline Backup Mode:** If your WiFi signal drops, check-ins are saved securely in the app's offline storage.
*   **Automatic Synchronization:** The app automatically sends any saved offline check-ins to the desktop workstation as soon as the connection is restored.

### 🔧 Quick Setup Guide

#### Reception Computer (Windows)
1. Download and run `SyncZen-Local-Setup-1.0.0.exe`.
2. When prompted by Windows, click **Allow / Yes** to grant permission for the app to communicate over the local WiFi network.

#### Staff Mobile Devices (Android)
1. Transfer the `SyncZen.apk` (or `app-release.apk`) to the Android phone or tablet.
2. Go to **Settings ➔ Apps ➔ Special app access ➔ Install unknown apps** on the device and enable permission for your file manager or browser.
3. Open your file manager app, find the APK file, and tap it to install.
4. Make sure both the computer and the mobile devices are connected to the **same WiFi network**.
